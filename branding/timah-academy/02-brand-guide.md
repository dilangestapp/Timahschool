# TIMAH ACADEMY — Mini Brand Guide

## Palette (HEX)
- Bleu nuit premium : `#0B1F4D`
- Bleu accent principal : `#1D6DFF`
- Bleu accent clair : `#7DB6FF`
- Blanc : `#FFFFFF`
- Texte/mono sombre : `#0E1726`

## Typographie recommandée
- Primaire : **Inter** (700 / 800)
- Alternative : **Manrope**
- Règle : **TIMAH** plus dominant visuellement que **ACADEMY** (poids + tracking)

## Charte d’usage rapide
- Navbar fond clair : `final-horizontal-light.svg`
- Navbar fond sombre : `final-horizontal-dark.svg`
- UI compacte / app / avatar : `final-icon.svg`
- Favicon : `favicon.svg`
- Impression / gravure : `final-monochrome.svg`

## Espace de protection
- Minimum recommandé : `0.5x` autour du logo (`x` = hauteur du bloc icône).

## Taille minimale
- Horizontal : 140 px de large
- Stacked : 96 px de large
- Icône : 24 px (UI), 16 px min (favicon)
